# Cerebro Landing Page

This directory contains the design for the Cerebro AI Research Platform landing page.

## Viewing the Page

Simply open `index.html` in any modern web browser.

## Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Styling (loaded via CDN for zero-build setup).
- **Vanilla JavaScript**: For the theme toggle and interactive terminal demo.
- **Google Fonts**: Plus Jakarta Sans.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Dark/Light Mode**: Toggles with a button and respects system preferences.
- **Interactive Demo**: Simulates a CLI research session.
- **Animations**: Subtle fade-ins and hover effects.
